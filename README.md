## ğŸ“Š Key Metrics Explained

### Portfolio KPIs
- **Total Value**: Sum of all stock holdings at current market price
- **Net Profit/Loss**: Difference between current value and initial investment
- **Return %**: Percentage gain or loss on total investment

### Volatility Metrics
- **Standard Deviation**: Measure of price fluctuation
- **Risk-Return Ratio**: Balance between potential gains and losses
- **Drawdown Analysis**: Maximum decline from peak value

### Contribution Analysis
- **Individual Stock Impact**: Each stock's contribution to overall portfolio performance
- **Top Performers**: Stocks with highest positive impact
- **Underperformers**: Stocks dragging down portfolio returns
